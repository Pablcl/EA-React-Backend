En este ejercicio he implementado un historial de acciones para usuarios y organizaciones

Cada vez que se crea, edita o elimina un usuario o organización se registra una acción.

He añadido una nueva vista en frontend (Actions) para consultar ese historial, y una nuevo model en backend (Accion) para guardarlo en base de datos.

Para probarlo, se puede hacer CRUD normal de usuarios/organizaciones y luego entrar en la pestaña Actions para ver los registros.

La IA se ha usado para:

- entender mejor el flujo completo frontend-backend
- ordenar la implementación por archivos y pasos
- apoyo en la creacion de componente, hook etc 
- ajustes de sintaxis en validaciones y TS

IA usada: ChatGPT 5.3
